# Practice partition

Supporting material for tasks whose manifests declare `"partition": "practice"`
(spec 13.2): solution explanations, expected traces, and instructor walkthrough notes
that students are allowed to see after attempting the task.

Current practice tasks (manifests live in `datasets/manifests/`):

- `bk-001` - Map the reservation lifecycle
- `bk-002` - Fix reservation list pagination
- `bk-007` - Fix the cancel-after-completion transition

Solution-explanation stubs land here module by module as the corresponding lessons are
published; nothing is published before the module that consumes the task. Until then this
directory intentionally contains only this README.
